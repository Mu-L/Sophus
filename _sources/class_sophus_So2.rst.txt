Group exponential.

This functions takes in an element of tangent space (= rotation angle ``theta``) and returns the corresponding element of the group SO(2).

To be more specific, this function computes ``expmat(hat(omega))`` with ``expmat(.)`` being the matrix exponential and ``hat(.)`` being the :ref:`hat() <doxid-classsophus_1_1_so2_1a36ea9ac1fca9e70a62ced6719052e773>` -operator of SO(2).

hat-operator

It takes in the scalar representation ``theta`` (= rotation angle) and returns the corresponding matrix representation of Lie algebra element.

Formally, the :ref:`hat() <doxid-classsophus_1_1_so2_1a36ea9ac1fca9e70a62ced6719052e773>` -operator of SO(2) is defined as

``hat(.): R^2 -> R^{2x2}, hat(theta) = theta * G``

with ``G`` being the infinitesimal generator of SO(2).

The corresponding inverse is the :ref:`vee() <doxid-classsophus_1_1_so2_1a668d6d544607a3d8eb587e37610b5339>` -operator, see below.

It takes the 2x2-matrix representation ``Omega`` and maps it to the corresponding scalar representation of Lie algebra.

This is the inverse of the :ref:`hat() <doxid-classsophus_1_1_so2_1a36ea9ac1fca9e70a62ced6719052e773>` -operator, see above.

Precondition: ``Omega`` must have the following structure:

	|  0 -a |
	|  a  0 |
