template struct sophus::IsFixedSizeVector
=========================================

.. toctree::
	:hidden:

If the Vector type is of fixed size, then IsFixedSizeVector::value will be true.


.. ref-code-block:: cpp
	:class: doxyrest-overview-code-block

	#include <types.h>
	
	template <
		typename VectorT,
		int kNumDimensions,
		typename = typename std::enable_if<VectorT::RowsAtCompileTime == kNumDimensions&&        VectorT::ColsAtCompileTime == 1>::type
		>
	struct IsFixedSizeVector: public true_type {
	};
