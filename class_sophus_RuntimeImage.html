Detailed Documentation¶

Type-erased image with shared ownership, and read-only access to pixels. Type is nullable.

Construction¶

RuntimeImage()

Empty image.

template <class PixelT>
RuntimeImage(Image<PixelT, AllocatorT> const& image)

Create type-erased image from Image.

Ownership is shared between RuntimeImage and Image, and hence the reference count will be increased by one (unless input is empty). By design not “explicit”.

template <class PixelT>
RuntimeImage(MutImage<PixelT>&& image)

Create type-erased image from MutImage. By design not “explicit”.

Methods¶

template <class PixelT>
bool has() const

Return true is this contains data of type PixelT.

template <class PixelT>
Image<PixelT, AllocatorT> image() const

Returns typed image.

Precondition: this-> has<PixelT>()

int numBytesPerPixelChannel() const

Number of bytes per channel of a single pixel.

E.g. a pixel of Eigen::Matrix<uint8_t, 3, 1> has 1 byte per channel.
